# Coisas que só a recepcionista pode fazer

- Marcar aula experimental
- Cadastrar aluno
- Vender plano
- Agendar aulas fixas
- Desmarcar aula
- Reagendar aula
- Alterar hora e dia de aula fixa.

# Coias que o admin pode fazer

- tudo o que a recepcionista faz
- cancelar plano
- Abrir agendas individuais. (uma por Studio)

# Coias que o instrutor pode fazer

- Ver agenda no qual está contratado
- Visualizar somente o nome do Aluno. ( não ter acesso
  aos dados pessoais).
- Reagendar aula
